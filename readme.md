Cornhole Pro 2.1 is an automatic scoring system for cornhole that uses computer vision, motion tracking, and configurable detection zones to reliably register points. The program works with both Picamera2 and standard webcams and is designed to run smoothly on Raspberry Pi or any desktop computer. After launching the script, the user selects the colors used by each player. These colors can optionally be calibrated automatically, allowing the software to measure the correct HSV ranges for the bags under current lighting conditions.
Before playing, the camera feed appears and the user draws two regions: first a rectangle around the board and then a circle around the hole. Both can be dragged afterwards for fine‑tuning. Once the match starts, Cornhole Pro continuously detects and tracks the bags, checks whether they are inside the board or hole area, and only awards points once a bag has come to rest. This prevents false triggers and ensures the scoring is accurate. A separate scoreboard window remains visible at all times, and the main interface provides large buttons for starting, pausing, resetting or manually adjusting scores.
To run the system, simply install the required Python packages (opencv-python, numpy, flask and optionally picamera2), place the script in a folder, and start it using python3 cornhole_pro.py. The interface guides you through selecting player colors, optional calibration, drawing the regions of interest and beginning the game. If desired, a built‑in Flask web interface can also be activated, allowing other devices on the network to view and control the scoreboard in real time.
Cornhole Pro 2.1 combines fast computer vision, a lightweight tracking engine and a clean interface to deliver a reliable automated scoring experience with live visuals, logging and remote control support.
